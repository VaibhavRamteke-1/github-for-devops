#  Git Learning & Real Project Workflow Guide

This repository contains practical Git concepts, real-world workflows, and commonly used commands explained with examples.

It is designed for:
- Beginners learning Git
- DevOps engineers handling CI/CD pipelines

##  Contents

-  git_commands.txt → Commonly used Git commands
-  git-branch-strategy.txt → Branching strategy (feature, develop, master, hotfix)
-  rebase-vs-merge.txt → Difference with diagrams
-  merge_conflict.txt → How to handle conflicts
-  revert-vs-reset.txt → Undo changes safely
-  git-real-project-workflow.txt → End-to-end real workflow

##  How to Use

1. Clone the repository
2. Go through each file step-by-step
3. Practice commands in your local Git setup
4. Apply in real projects
